Skills FrontEnd

O HTML é peça fundamental da criação de sites. Ele é quem dá todo o esqueleto das páginas e define o que vai nelas.

Da mesma forma que o HTML é o esqueleto o CSS é a pele. O CSS define como os elementos aparecem na página

Com o esqueleto e pele prontos, só falta darmos vida ao site. O Javascript são os musculos que permitem ao usuário interagir com elementos da página.

React é uma biblioteca muito útil que facilita a interação do HTML, CSS e JS. Com ele, sempre estamos "repensando boas práticas".
